# Snack Overflow — Diet Manager

Web application for planning meals around personal nutrition targets. Set daily calorie & macro goals, search recipes, swap ingredients, and get suggestions based on what you have in your kitchen.

## Contents

- **`lofi-prototype.html`**: Low-fidelity wireframes; append `?frame=N` (1–6) to view a single frame.
- **`lofi-screens/`**: PNG exports of each lo-fi wireframe (1-homepage through 6-result).
- **`prototype.html`**: Interactive hi-fi prototype.
